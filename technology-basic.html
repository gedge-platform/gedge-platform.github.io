GEdge Platform 개념

GEdge 플랫폼 이란?

GEdge Platform(Griffin-Edge Platform)은 응답 속도 민감형 서비스를 위해, 다양한 클라우드 및 엣지 컴퓨팅 인프라를 연동하여 대규모 확장성을 갖춘 클라우드 엣지 컴퓨팅 인프라를 구성하고 이를 기반으로 코어 클라우드-클라우드 엣지-단말간 서비스의 분산 협업을 통한 초저지연 데이터 처리를 지원하는 클라우드 엣지 SW 플랫폼이다.

클라우드 엣지 컴퓨팅 : 엣지에서 클라우드 서비스 기반의 컴퓨팅 환경을 제공하고 클라우드-엣지-단말, 엣지-엣지간 협업을 통한 지능형 서비스를 제공하는 분산 클라우드 컴퓨팅 방식

GEdge 플랫폼 핵심 요소 기술

GEdge 플랫폼 개념도

GEdge 플랫폼은 응답 속도 민감형 서비스를 위해, 코어 클라우드-클라우드 엣지-단말 간 협업 기반 초저지연 데이터 처리를 지원한다.

GEdge 플랫폼 기능 프레임워크 & SW 스택

GEdge 플랫폼은 2개의 핵심 플랫폼 및 5개의 기능 프레임워크로 구성되며 인프라 레이어부터 서비스 레이어까지의 구성 및 관리 기능을 제공한다.

  • GS(Gedge Service) 플랫폼 : 대규모 엣지 데이터 저지연 처리 및 서비스 협업 통한 지능형 서비스 최적화 제공 서비스 플랫폼
    • GS-Engine : 초저지연 데이터 처리 프레임워크
    • GS-Link : 수직-수평적 서비스 협업 프레임워크
    • GS-AI : 지능형 서비스 운용 프레임워크
  • GM(Gedge Management) 플랫폼 : 가변형 클라우드 엣지 플랫폼 구성/관리 및 워크스페이스 기반 관리 도구 제공 관리 플랫폼
    • GM-Center : 플랫폼 관리 프레임워크
    • GM-Tool : 플랫폼 관리 도구 프레임워크

GEdge 플랫폼 아키텍쳐

GEdge 플랫폼은 정의된 SW 스택을 기반으로 코어 클라우드-클라우드 엣지간의 서비스 협업을 지원하고 서비스 개발자 및 운영자에게 QoE 보장 서비스 배포 기능을 제공한다.
